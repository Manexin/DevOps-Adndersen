About this app

This application uses the POST and GET methods. Receives a word at the input and repeats this word the specified number of times at the output and decorates it with emojis. 😎
In the task, it was said that the application should decorate the line with my favorite emojis, but unfortunately the `Emoji 1.2` module does not support all emojis, so I chose a few that I liked that the module supports.
You can access the application as follows:
curl -XPOST -d'{"word":"flower", "count": 10}' http://your_server:8080/ Or you can open the page in a web browser at http://your_server:8080 Of course, instead of your_server, you need to enter the address of the server where you installed my application 😁 although if you are reading this text, then you did everything right. 👍
Enjoy 👀
